Altcoin Voting

Vote for which altcoin(s) you would like to be researched in the selected month's Altcoin Investigation book

Select any altcoin below to vote for it to be included in the next edition of the selected month's Altcoin Investigation. If the selected month is the current month, then editions come out daily, if possible. Note that you can vote for as many coins as you want, as many times as you like, to accurately reflect your desire to receive information about the coin. There may be a 24-hour delay between the casting of votes and resulting additions to the next edition. Altcoins receiving the highest number of votes will be the ones included. If an altcoin button is disabled and the row has a grey background, then research on the altcoin has already been included in the book.
